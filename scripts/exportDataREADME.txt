exportData.js README

exports all orders and costs for given time period

SETTINGS: open script in any editor and set:

for 'From' Date: dayFrom, monthFrom*, yearFrom
for 'To' Date: dayTo, monthTo*, yearTo

month is numberes 0-11 in javascript instead of 1-12!

valid Date example:
dayFrom: 10,
monthFrom: 5,
yearFrom: 2016
                    => 10. June, 2016

EXECUTION: node exportData.js*
* from git bash in script directory
